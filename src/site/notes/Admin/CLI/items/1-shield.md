# +1 Shield
*Shield, major, uncommon*  

- **Armor Class**: 2
- **Weight**: 6.0 lbs.

While holding this shield, you have a +1 bonus to AC. This bonus is in addition to the shield's normal bonus to AC.

*Source: Dungeon Master's Guide p. 200. Available in the SRD.*
